IWD:EE portrait pack: Portraits from Icewind Dale II

Version: 01
Release date: Jan 2021
Links: forum thread • github
Mod author: Ineth
Image artists: Justin Sweet, and possibly other artists at Black Isle Studios back in 2002.

Introduction

This portrait pack consists of the 24 official Icewind Dale II portraits that are not already included in Icewind Dale: Enhanced Edition.

(I have remastered them to the resolution and aspect ratio required by IWD:EE. Check the forum link above for more info.)

Installation

As a mod

This portrait pack can be installed the same way as most IWD:EE mods:

  • Instructions for Windows
    1. Find your Icewind Dale: Enhanced Edition game directory.
      This depends on how you installed the game. Some possible locations are:
      Steam version: C:\Program Files (x86)\Steam\SteamApps\common\Icewind Dale Enhanced Edition\
      GOG version: C:\GOG Games\Icewind Dale Enhanced Edition\
    2. Extract this mod into the game directory.
      When done correctly, your game directory will contain the file setup-PortraitsFromIWD2.exe and the folder PortraitsFromIWD2.
    3. Double-click setup-PortraitsFromIWD2.exe and follow the instructions on screen.
      (To help you decide between the normal vs zoomed-in side-bar portraits, refer to the Portrait layouts section below.)

    You can run setup-PortraitsFromIWD2.exe again afterwards to uninstall or reinstall the mod.
  • Instructions for macOS
    1. Find your Icewind Dale: Enhanced Edition game directory.
      This depends on how you installed the game, and can be a bit difficult to navigate to. See step 1 of this guide for help.
    2. Extract this mod into the game directory.
      When done correctly, your game directory will directly contain the folder PortraitsFromIWD2.
    3. Download the MacOS Weidu Launcher, unzip it, and drop the .app into your game directory and double-click it there. Then, follow the instructions on screen.
      (To help you decide between the normal vs zoomed-in side-bar portraits, refer to the Portrait layouts section below.)
  • Instructions for Linux

    Making your Linux installation of IWD:EE ready for WeiDU-based mod installers, takes a bit of work – so you may find it less time-consuming to use the manual installation method for portraits instead.

    If you do want to install this portrait pack as a mod, follow this guide.

There are no known incompatibilities or install order requirements relative to other mods.

Manual installation

Alternatively, you can manually copy the portraits into your user account's portraits folder, where the game will find them:

  • Instructions
    1. Find your IWD:EE user portraits folder:
      Windows & macOS: Documents/Icewind Dale - Enhanced Edition/Portraits
      Linux: .local/share/Icewind Dale - Enhanced Edition/portraits/
      (The paths shown are relative to your user account's home directory. You have to start the game at least once, for them to be created.)
    2. Copy all the .bmp files from the L subfolder of this download, to that location.
    3. Copy all the .bmp files from the M subfolder or from the M_zoomed subfolder to that location as well.
      (Refer to the Portrait layouts section below to help you choose.)
    4. If you have patch version 2.x of the game, the portraits now appear in the in-game portrait selection list, after the original portraits. (However, unlike when installing the portrait pack as a mod, they're not separated by gender - all portraits show up for both male and female characters.)
      If you have patch version 1.x of the game, a "Custom portraits" button instead appears on the in-game portrait selection screen, allowing you to pick them.

Portrait layouts

This portrait pack contains three .bmp files per portrait – one for each of these layouts:

L M M_zoomed

Character sheet portrait

Cropped like the built-in ones.

Side-bar portrait

Cropped like the built-in ones.

Side-bar portrait

Cropped differently from the built-in ones:

  • Zoomed in more closely on the face.
  • Flipped where necessary to make sure no character faces right
    (because that's where the screen edge is).

It's up to you whether you want to use this instead of the standard M layout. (See the Installation section above.)

If you want the built-in portraits to use this layout as well, install the "Zoomed-in side bar portraits" component of my IWD:EE Portrait Variations mod.
